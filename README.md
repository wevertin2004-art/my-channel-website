# ⚡ Wevertin | Official Channel Website

This is the high-end, minimalist portfolio website for the **Wevertin** YouTube channel. Designed with a "Million Dollar" aesthetic, featuring smooth scroll animations, a bilingual toggle (EN/PT), and a fully responsive layout.

## 🚀 Features
* **Bilingual Support:** Instant English/Portuguese translation without page reloads.
* **Modern Stack:** Built with pure HTML5, CSS3 (Custom Properties), and Vanilla JavaScript.
* **Performance:** Ultra-lightweight with high SEO scores and smooth 60fps animations.
* **Responsive:** Optimized for 4K monitors, Laptops, Tablets, and Smartphones.

## 📺 Content
The site highlights the latest gaming tutorials and high-tier gameplay from [__wevertin__ on YouTube](https://www.youtube.com/@__wevertin__).

## 🛠️ How to Update Videos
To add new content to the site:
1. Open `index.html`.
2. Locate the `<div class="grid">` section.
3. Copy a `video-card` block and update the `href`, `img src`, and `h3` title.
4. Push the changes to GitHub to auto-deploy.
